Purpose:
A program that allows a user to encrypt and decrypt a message (from a file, to a new file). 


Dependencies:
Requires the installation of cryptography. Can be done using command:  pip install cryptography

Collected packages: pycparser, cffi, cryptography
At time of writing, versions:  cffi-1.14.5 cryptography-3.4.7 pycparser-2.20


Instructions:
- Run the program, passing either "e" or "d" to indicate encrypt or decrypt.
- Enter the cryptography secret key when prompted. It is "3ya3VRFu9SNtETmlfAbYAhhf4fqBGA5fBbWmEyQ3QSE=".
- Enter the filename when prompted.
- If successful, a new file will be created and the name displayed to the user.
